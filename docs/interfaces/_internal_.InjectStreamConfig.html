Optional audioSampleRate

audioSampleRate?: number
  • Defined in node_modules/agora-rtc-sdk-ng/rtc-sdk_en.d.ts:2897

The audio sampling rate of the injected stream.

  • 44100: (Default) 44.1 kHz
  • 48000: 48 kHz
